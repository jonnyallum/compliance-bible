# Release Notes

## Version 0.1.0

Release date: 2026-06-24

## Summary

This release establishes the Compliance Bible as a production-ready baseline for UK FM compliance content, FM Control Hub product mapping, Compliance Hub evidence workflows and Longleat interview preparation.

## Included

- Core compliance pages for major FM legal duties.
- Operational frameworks for risk assessment, contractor management and permit-to-work.
- Statutory compliance matrix.
- FM Control Hub schema and Supabase seed data.
- 50-template FM operational library.
- Audit checklist library.
- Evidence retention policy.
- Navigation, tags, glossary, quick start and roadmap.
- AI/RAG metadata across key content.
- Quality audit and fix list.

## Fixes in This Release

- Corrected Supabase schema index creation order.
- Corrected README folder naming for the permit-to-work framework.
- Added missing AI/RAG metadata to content documents.

## Known Limitations

- Some library files are long for phone-first reading and should be split later.
- SQL has not been executed against a live Supabase database in this audit pass.
- Remaining TODO items are intentionally left incomplete until production content exists.
