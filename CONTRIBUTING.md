# Contributing

## Production Standard

This repository is intended to become a reliable operational compliance source for Facilities Management and compliance software.

Do not add:

- placeholder pages
- half-written modules
- uncited legal claims
- vague advice
- duplicated filler content
- empty folders

## Source Standard

Use official or authoritative sources first:

- HSE
- GOV.UK
- legislation.gov.uk
- Home Office
- Environment Agency
- CQC where relevant
- recognised professional bodies where official guidance is not available

## Required Review Checklist

Before committing, confirm:

- [ ] The page is complete.
- [ ] The legal or guidance source is named.
- [ ] Official links are included.
- [ ] FM duties are explained in practical language.
- [ ] Evidence requirements are listed.
- [ ] Review or inspection frequency is included where relevant.
- [ ] FM Control Hub mapping is included.
- [ ] Compliance Hub mapping is included.
- [ ] `TODO.md` is updated if a task is completed.
- [ ] `CHANGELOG.md` is updated.

## Page Quality Test

A page is good enough only if a Facilities Manager could use it to answer:

1. What am I responsible for?
2. What do I need to check?
3. How often should I check it?
4. What evidence do I need to keep?
5. What happens if this fails?
6. How should this be represented in FM Control Hub or Compliance Hub?
